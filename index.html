Python এর ডেটা টাইপ (Data Type) মূলত ৫ প্রকারের। নিচে একটি ছকের মাধ্যমে সেগুলোর উদাহরণ সহ দেখানো হলো:

ডেটা টাইপ বর্ণনা উদাহরণ
Numeric সংখ্যাসমূহ নিয়ে কাজ করে int, float, complex
Sequence বিভিন্ন ডেটার ধারাবাহিকতা list, tuple, range
Text Type টেক্সট বা স্ট্রিং তথ্যের জন্য str
Boolean সত্য/মিথ্যার মান প্রদর্শন করে True, False
Set অনন্য ডেটার অসংকলন set, frozenset
Mapping key-value পদ্ধতিতে তথ্য রাখে dict
